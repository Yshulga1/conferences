## File Types
1. **PNG:**
  * Transparency
  * Higher quality
  * PNG8: Similar to GIF
  * PNG24: Many colors, larger images
2. **JPEG/JPG:**
  * Complex algorithm for file compression
  * Best use: still images, photos, complexity
3. **GIF:**
  * Lower number of colors
  * Reuses colors throughout the image
  * Best use: few colors, animations
